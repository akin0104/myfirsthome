# MyFirstHome – Starter Home Planner

## Description
MyFirstHome is a simple web app that helps users plan their first home by saving properties, prices, and personal notes. Users can mark favourites and compare options easily.

## Features
- Add property (name + price)
- Add personal notes
- Mark as favourite
- Delete property
- Filter favourites
- Summary dashboard (total, highest, lowest)
- Data saved using localStorage
- Property images

## Tech Used
- HTML
- CSS
- Vanilla JavaScript
- LocalStorage
